# test_nmea_reading

Flutter app (iOs+Android) for test ble gps nmea reading

## what it is

Bluetooth NMEA receiver + tiny nmea dart parser

Tested with AT-09 (HM-10, BT05, BT-05) BLE module

## wiring

VCC, GND - to power supplier

GPS TX pin to Bluetooth RX pin

That is all!
